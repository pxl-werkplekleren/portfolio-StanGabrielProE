Ontwerper

Waar sta ik momenteel?

Ik begrijp de basis van analyse en ontwerp, en ik kan een eenvoudige structuur voor een project uittekenen. Ik weet hoe ik klassen, methodes en logische stappen moet plannen voordat ik begin te programmeren.

Wat lukt al goed?

Ik kan een probleem opdelen in kleinere delen en een overzicht maken van wat er nodig is. Ik denk logisch en gestructureerd, wat helpt bij het voorbereiden van een project.

Wat lukt nog niet?

Ik moet nog groeien in het maken van meer gedetailleerde ontwerpen en het kiezen van de juiste methodiek voor grotere projecten. Ook wil ik leren hoe ik mijn ontwerp beter kan afstemmen met anderen.

Programmeur

Waar sta ik momenteel?

Ik beheers de basis van C# en kan console en WPF toepassingen bouwen. Ik begrijp variabelen, controle structuren, methodes en eenvoudige objectgeoriënteerde concepten.

Wat lukt al goed?

Ik schrijf duidelijke en gestructureerde code. Ik kan fouten opsporen en oplossen, en ik leer snel nieuwe concepten.

Wat lukt nog niet?

Ik moet nog oefenen met complexere OOP concepten zoals interfaces, inheritance en grotere projectstructuren. Ook wil ik mijn code nog consistenter documenteren.

Tester

Waar sta ik momenteel?

Ik begrijp het belang van testen en kan mijn eigen code controleren met test scenario's en debugging tools.

Wat lukt al goed?

Ik kan fouten logisch analyseren en oplossen. Ik gebruik debugging in Visual Studio efficiënt en ik kan feedback toepassen om mijn code te verbeteren

Wat lukt nog niet?

Ik heb nog weinig ervaring met formele test methodes en geautomatiseerd testen. Ik wil leren hoe ik betere testcases kan opstellen.

Communicator / Teamspeler

Waar sta ik momenteel?

Ik communiceer duidelijk en professioneel, zowel schriftelijk als mondeling. In het Engels gaat dit al zeer vlot, maar in het Nederlands (Vlaams) ben ik nog volop aan het groeien. Ik kan samenwerken en vragen stellen wanneer dat nodig is, maar ik merk dat mijn communicatie sterker en spontaner is wanneer ik Engels gebruik.

Wat lukt al goed?

Ik kan feedback ontvangen en verwerken, ongeacht de taal. Ik leg mijn werk gestructureerd uit en ik kan anderen helpen wanneer ze vastzitten. Vooral in het Engels kan ik mijn ideeën helder en precies formuleren, wat mij helpt om efficiënt samen te werken.

Wat lukt nog niet?

Ik wil nog groeien in het actief deelnemen aan groepsgesprekken in het Nederlands. Het presenteren van technische informatie aan een niet-technisch publiek lukt mij goed in het Engels, maar in het Vlaams moet ik nog oefenen om even vlot en zelfzeker te klinken. Ik werk eraan om mijn woordenschat en spreekvaardigheid verder te verbeteren.

Levenslang lerende IT‑professional

Waar sta ik momenteel?

Ik ben gemotiveerd om voortdurend bij te leren en ik volg nieuwe technologieën op. Ik reflecteer regelmatig op mijn eigen werk

Wat lukt al goed?

Ik leer zelfstandig, zoek oplossingen op en verbeter mijn vaardigheden stap voor stap. Ik ben kritisch voor mijn eigen werk en probeer altijd beter te worden.

Wat lukt nog niet?

Ik moet nog leren hoe ik mijn leerproces beter kan plannen op lange termijn en hoe ik nieuwe kennis systematisch kan toepassen in projecten.
