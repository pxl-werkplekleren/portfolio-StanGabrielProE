Goesting

Tijdens Werkplekleren 1 merkte ik dat mijn motivatie groeide telkens wanneer ik nieuwe dingen ontdekte. Vooral de eerste stappen in C#, het bouwen van mijn console‑applicatie en later het werken met WPF gaven mij echt energie. Ik kreeg goesting om verder te leren omdat ik meteen resultaat zag van mijn inspanningen. Ook het maken van mijn portfolio en CV zorgde voor extra motivatie: ik voelde dat ik bezig was met iets dat echt bijdraagt aan mijn toekomst als programmeur.

Competentie

Mijn competenties zijn sterk gegroeid door de verschillende opdrachten. Ik leerde werken met Git & GitHub, ik bouwde meerdere C#‑toepassingen en ik verbeterde mijn debugging‑vaardigheden. Door feedback te verwerken, begreep ik beter hoe ik mijn code kan optimaliseren. Ook HTML en CSS hebben mijn technische basis verbreed. Elke opdracht hielp mij om stap voor stap meer vertrouwen te krijgen in mijn vaardigheden als programmeur.

Autonomie

Tijdens dit semester heb ik geleerd om zelfstandiger te werken. Ik kon opdrachten plannen, problemen analyseren en oplossingen zoeken zonder altijd hulp te vragen. Vooral bij debugging en WPF merkte ik dat ik steeds vaker zelf tot een oplossing kwam. Het maken van mijn portfolio vroeg ook veel autonomie: ik moest zelf beslissen hoe ik alles structureerde en vormgaf. Deze zelfstandigheid heeft mij sterker gemaakt als student én als toekomstige ontwikkelaar.

Verbondenheid

Hoewel ik graag zelfstandig werk, heb ik tijdens Werkplekleren ook ervaren hoe belangrijk samenwerking is. Ik stelde vragen wanneer ik vastzat, ik luisterde naar feedback van lectoren en ik deelde mijn ervaringen met medestudenten. Vooral de sessies met gastsprekers en groepsmomenten rond POP zorgden voor verbondenheid. Ik voelde dat ik deel uitmaak van een groep studenten die hetzelfde doel heeft: groeien in IT. Dat gaf mij vertrouwen en motivatie.
